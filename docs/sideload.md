# Sideload

1. [Download the Build Add-In App Package](https://wandyezj.github.io/build-add-in/manifests/appPackage.zip)

2. [Upload the App Package in Teams](https://learn.microsoft.com/en-us/microsoftteams/platform/concepts/deploy-and-publish/apps-upload#upload-your-app)


note: If you are using a managed account, an IT setting can disable loading from the store and a setting that can disable sideloading. Even if you can't access the store you might still be able to sideload.
