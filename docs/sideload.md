# Sideload the Build Add-In in Teams

1. [Download the Build Add-In app package](https://wandyezj.github.io/build-add-in/manifests/appPackage.zip).

2. [Upload the package in Teams](https://learn.microsoft.com/en-us/microsoftteams/platform/concepts/deploy-and-publish/apps-upload#upload-your-app).


If you use a managed account, your IT admin may block store installs or sideloading. If the store is blocked, sideloading may still be available.
